1. 사용자의 분석 요청에 supabase에 이미 있어서 캐시된 데이터를 조회하더라도, 사용자에게 캐시 데이터를 조회했다는 안내메시지는 보여주지마. 그리고 내정보의 분석가능 횟수도 캐시 데이터 조회 여부와 상관없이 항상 1씩 차감시켜줘
2. 내정보 화면의 분석 가능 횟수가 0이 되면 분석가능 횟수 카드뷰 내에 '분석가능횟수 충전' 버튼을 표시해줘
3. '분석가능횟수 충전' 버튼을 누르면 분석가능 횟수를 충전을 위한 페이지로 이동해줘
4. '분석가능횟수 충전' 페이지에는 20회: 500원, 50회: 1000원 충전 플랜으로 구성해줘
5. 구매 플랜을 선택하면, 토스 페이먼츠 결제 연동 기능을 통해서 결제를 할 수 있게 해줘
    - 가이드 문서 : https://docs.tosspayments.com/guides/v2/payment-widget/integration
